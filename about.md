# 关于本项目

全球人工智能治理法律与政策跟踪平台旨在汇集世界主要经济体与具有代表性的国家关于人工智能治理的法律、行政法规、政策文件和技术出口管制措施，方便学术界和实务界开展比较研究。本平台由研究者发起，并依托于 GitHub Pages 部署。

## 项目目标

- 跟踪各国人工智能监管框架的最新动态。
- 提供标准化的数据结构，便于学者进行分析。
- 促进人工智能治理领域的学术交流与合作。

## 数据来源与收录标准

数据主要来源于各国官方立法数据库、监管机构发布的政策文件、国际组织报告以及可信的媒体报道。平台对收录的文件进行分类整理，并使用人工智能工具生成摘要与标签。

## 更新周期

系统每周自动检查各国官方渠道发布的与人工智能相关的法规和政策。一旦检测到更新，平台会自动更新数据并生成对应的网页内容。
